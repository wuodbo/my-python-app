#  Movie Database Management System

Интерактивная система управления базой данных фильмов с пользовательским интерфейсом на Python.

## Основные возможности

### Режим базы данных
- Управление таблицами фильмов
- Управление сценами
- Управление решениями
- Управление зрителями
- Управление картами просмотров
- Управление сюжетами
- Управление сценариями

### Dashboard режим
- Мониторинг активности пользователей
- Статистика просмотров
- Часть для общения пользователей
- Навигационная панель

### 1. Предварительные требования
- Python 3.8 или выше
- MySQL сервер
